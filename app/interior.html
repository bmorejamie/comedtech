Comedtech Data Trust

Data Trust is our game changing data analytics platform. Aggregation, search, and analysis all available through a simple to use natural language search interface.

All your data in one location: Searchable…Secure…Simplified.

Data Aggregation

Trust Translators are data codecs that query, retrieve, and translate data from systems large and small. With more than 170 Trust Translators available today, access to any data type and any system is fast, simple, and secure.

Once a real time data feed has been established via the correct Trust Translator, data is automatically aggregated physically or virtually into Trust Core. Trust Core is the central secure repository for all data feeds. Data is then indexed and made searchable.

Existing structured data systems including EMR and CRM systems can also easily be included in Trust Core. Trust Translators already exist for the most popular EMR and CRM systems available today.

Data Search

A natural language search interface queries data in Trust Core and provides instant results. Aggregated and multiple step queries are supported and can be saved as well as shared with colleagues.

Data Security

By design, all data in Trust Core is automatically anonymized and de-identified. Results provided are pseudo-random data controlled by Trust Engine.

With Trust Engine, precise control over data access can be set, audited, and monitored in real time. For properly credentialed employees, access to identifiable data can be easily provided to support clinical care delivery workflows.

Deployment Options

Data Trust can be deployed as either a physical or virtual system. It easily can scale from a single department to enterprise wide.

How to Evaluate

Data Trust can be evaluated for your organization today by contacting us for more information. Typical deployment time for a department is two weeks. Depending on data size, initial processing can take days or weeks. After initial processing, real time results are available from all Trust Translators.
